# Privacy

The default design processes media in memory and does not retain video or audio. Evidence stores aggregate counters, hashes, versions and bounded diagnostic markers. It must not contain gameplay recordings, packet payloads, account names, tokens, passphrases, cookies, precise public IP addresses, or unrelated desktop content.

The PWA keeps session state in memory. Persistent browser storage is limited to non-secret UI preferences and the service-worker cache. Telemetry upload is disabled unless explicitly configured and documented.
